Бот используется для рассказа о проблеме мусора.

Есль команды:
/start - запускает бота
/help - выдаёт основные команды
/graz - даёт выбор из функций.
/problema - Коротко о проблеме.
/dengi - Сколько можно получить за сдачу мусора.
/vreme - Сколько мусор будет разлогаться.

С работой бота можно ознакомиться по ссылке:
